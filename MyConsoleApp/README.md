# MyConsoleApp

This is a simple C# .NET console application that prints "Hello, World!" to the console. 

## Getting Started

To run this application, you need to have the .NET SDK installed on your machine. 

## Running the Application

1. Open a terminal and navigate to the project directory.
2. Run the following command to build the application:
   ```
   dotnet build
   ```
3. After building, run the application with:
   ```
   dotnet run
   ```
